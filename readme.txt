== Installation ==

1. Copiar el contenido del folder cambio_monedas a '/wp-content/plugins/cambio_monedas'
2. Activar el plug-in
3. Acceder al plug-in usando el menú Cambio de Monedas.


== Frequently Asked Questions ==

= ¿Cuál es la fuente de datos? =

Se usa http://free.currencyconverterapi.com, no se especifica la fuente de datos, si es útil, por favor compra en https://www.currencyconverterapi.com/pricing

= ¿Existe un límite en la cantidad de conversiones? =

Sí, como se utiliza la versión gratuíta, está limitado a 100 conversiones por hora
